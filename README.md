## فارسی

### معرفی پروژه
این پروژه یک ابزار تعاملی و آموزشی برای تحلیل **درخت پوشای مینیمم (MST)** و مقایسه الگوریتم‌های **کراسکال (Kruskal)** و **پریم (Prim)** است. این ابزار به صورت دوزبانه طراحی شده و قابلیت اجرا در ویندوز، اندروید و وب را دارد.

### ویژگی‌های کلیدی
- **دوزبانه:** پشتیبانی کامل از زبان‌های فارسی و انگلیسی.
- **خروجی PDF:** تولید گزارش شیک و تمیز در حالت Landscape.
- **نسخه اندروید (PWA):** قابلیت نصب روی گوشی و استفاده آفلاین.
- **نسخه ویندوز (EXE):** دارای لانچر اختصاصی بدون نیاز به پیش‌نیاز.
- **کاملاً آفلاین:** تمام کتابخانه‌ها (MathJax, Fonts) به صورت محلی در پروژه قرار دارند.

### نحوه استفاده
1. **وب:** پروژه را در GitHub Pages آپلود کرده و لینک را باز کنید.
2. **ویندوز:** فایل `main.exe` را از پوشه `dist` اجرا کنید.
3. **اندروید:** لینک سایت را در کروم باز کرده و گزینه "Install App" را بزنید.

## English

### Project Overview
An interactive educational tool for analyzing the **Minimum Spanning Tree (MST)** and exploring **Kruskal's** and **Prim's** algorithms. Designed for students and researchers with a focus on usability and accessibility.

### Key Features
- **Bilingual Support:** Full Persian and English localization.
- **Professional PDF Export:** Clean, landscape-oriented reports.
- **Android Support (PWA):** Installable on mobile devices with offline capabilities.
- **Windows Launcher (EXE):** Self-contained executable for easy local access.
- **100% Offline:** All dependencies (MathJax, fonts, libs) are bundled locally.

### How to Run
1. **Web:** Deploy to GitHub Pages and open the provided URL.
2. **Windows:** Run `main.exe` from the `dist` folder.
3. **Android:** Open the URL in Chrome and select "Install App".
